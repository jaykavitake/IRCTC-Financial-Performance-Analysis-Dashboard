# IRCTC-Financial-Performance-Analysis-Dashboard
An interactive Excel dashboard analyzing IRCTC's financial performance from 2015 to 2025, built as part of my Data Analytics learning journey.


# Project Overview
This project presents a comprehensive financial performance analysis of IRCTC (Indian Railway Catering and Tourism Corporation) — one of India's most prominent public sector enterprises — spanning 10 years (2015–2025).

The dashboard was built entirely in Microsoft Excel using Pivot Tables, dynamic charts, slicers, and KPI cards. It is designed to give a clear, at-a-glance view of IRCTC's revenue growth, profitability trends, operational efficiency, and dividend behavior over the years.

This project marks a significant milestone in my Data Analytics journey, demonstrating skills in data cleaning, financial analysis, and dashboard design.

# Dataset Description
The dataset covers IRCTC's annual financial statements with the following key fields:
FieldDescriptionYearFinancial year (Mar-2015 to Mar-2025), RevenueTotal revenue earned (in Crores), ExpensesTotal expenses incurred (in Crores), Net ProfitProfit after tax (in Crores), Other IncomeNon-operating income (in Crores), EPSEarnings Per Share , Dividend % Dividend payout percentage , OPM % Operating Profit Margin percentage , Tax % Effective tax rate percentage
